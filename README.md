# Instagram Catalog Website

Learning project to build a catalog website for an Instagram-based product business.

## Goals
- Learn Java by building a real project
- Learn Spring Boot for backend APIs
- Learn React for frontend
- Learn AWS deployment basics
- Use Codex to assist with implementation

## Project Structure
- backend: Java + Spring Boot API
- frontend: React app
- docs: notes and architecture
- infra: deployment notes and configs
- prompts: reusable Codex prompts
